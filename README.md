# Honeymoon Planner

모바일에서 보기 좋은 신혼여행 계획표와 체크리스트를 담은 단일 HTML 프로젝트입니다.

배포 링크: [https://wani3000.github.io/honeymoon/](https://wani3000.github.io/honeymoon/)

## 소개

이 프로젝트는 리스본, 마요르카, 마드리드 신혼여행 일정을 한 화면에서 보기 쉽게 정리한 페이지입니다.  
일정표, 예약 체크, 여행 회화까지 모바일에서 바로 확인할 수 있도록 구성했습니다.

## 주요 기능

- 날짜별 일정표
- 시간별 이동 방식 표시
- 위치 복사 버튼과 Google Maps 바로가기
- 진행률이 반영되는 여행 체크리스트
- 스페인어 / 포르투갈어 기초 여행 회화
- 모바일 반응형 UI
- 다크모드 대응
- 체크리스트 상태 `localStorage` 저장

## 파일 구성

- `index.html`: 메인 페이지
- `styles.css`: 보조 스타일 파일
- `app.js`: 보조 스크립트 파일
- `assets/route-overview.svg`: 경로 이미지
- `config.example.js`: 설정 예시 파일

현재 실제 화면은 `index.html` 중심의 단일 페이지로 동작합니다.
